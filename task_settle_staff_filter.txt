## 수정 목표
① 담당자별 집계의 "정산담당자 기준" 집계를
더보상 TF 소속 건(tfName.startsWith('더보상'))으로만 필터링

---

## 수정 파일
app/branch/_components/PerformanceTab.tsx

## 수정 내용

기존 settleStaffSummary 집계 로직을 찾아서:

### 기존:
```typescript
const settleStaffSummary: Record<string, { count: number; totalGross: number; incentive: number }> = {}
for (const s of filteredSettlements) {
  for (const a of (s.allocations || [])) {
    const name = a.staffName || '(미상)'
    if (!settleStaffSummary[name]) settleStaffSummary[name] = { count: 0, totalGross: 0, incentive: 0 }
    settleStaffSummary[name].count      += 1
    settleStaffSummary[name].totalGross += Math.floor(s.grossAmount * a.ratio / 100)
    settleStaffSummary[name].incentive  += Math.floor(s.grossAmount / 1.1 * (a.ratio / 100) * 0.1)
  }
}
```

### 교체:
```typescript
// 정산담당자 기준 집계 — 더보상 TF 소속 건만 대상
const settleStaffSummary: Record<string, { count: number; totalGross: number; incentive: number }> = {}
for (const s of filteredSettlements) {
  if (!s.tfName?.startsWith('더보상')) continue  // 이산 TF 제외
  for (const a of (s.allocations || [])) {
    const name = a.staffName || '(미상)'
    if (!settleStaffSummary[name]) settleStaffSummary[name] = { count: 0, totalGross: 0, incentive: 0 }
    settleStaffSummary[name].count      += 1
    settleStaffSummary[name].totalGross += Math.floor(s.grossAmount * a.ratio / 100)
    settleStaffSummary[name].incentive  += Math.floor(s.grossAmount / 1.1 * (a.ratio / 100) * 0.1)
  }
}
```
