## 4. 페이지별 UI 스펙

### 4-1. 로그인 페이지
- 상단: 대표 인사말 박스(좌) + 관리자 공지사항 박스(우)
- 하단: 로그인 입력창 (가로로 넓게, 중앙 정렬)
- [가입 요청] 버튼: 로그인 버튼 하단

### 4-2. To Do List
- 좌측 1/2: 달력 그리드 + 날짜별 To Do 이력
- 우측 1/2 상단: 요청 유형별 필터 버튼
- 우측 1/2 하단: 기간 설정 조회
- 하단 여백: 자유 기술 메모 공간

### 4-3. 사건 목록 (`/cases`)
- 상단: 다중 필터 (상병, 진행단계, 담당자, TF, 지사)
- 테이블: 이름, 주민번호, 상병, 진행상황(한글), 담당자, TF, 수임일
- 진행상황: 한글 로컬라이징 적용 (영문 Enum 표시 금지)
- 행 클릭: 사건 상세로 이동

### 4-4. 사건 상세 (`/cases/[caseId]` 또는 `/patients/[id]`)
- 상단: 재해자 기본 정보 (이름, 주민번호, 연락처, 주소)
- 탭 구성: 사건 정보 / 직업력 / 특진 / 전문조사 / 결정 / 이의제기
- 타임라인: 우측 or 하단 (진행 이력 표시)
- 하단: 보험급여 서식 생성 버튼

### 4-5. 직업력 테이블
- 근무기간 컬럼 표시
- 일용직 20일 미달 시 갈색 강조 (`isDailyWorker` 연동)

### 4-6. 특진 (HearingLossExam)
- 기본: 1~3차 + +N차 버튼으로 동적 추가
- 재특진: 기본 숨김 → 필요 시 펼치기
- 각 회차: 검사일, 담당자, 청력 수치 (기도/골도/어음), ABR, 임피던스, 신뢰성

### 4-7. 상담 관리 (수임 상태 배지)
| 상태 | 색상 |
|------|------|
| 수임 | 스카이 블루 |
| 미수임 | 그레이 |
| 보류 | 그레이 (2026.03.23 추가) |
| 기타 | 기본 |

### 4-8. 지사장 관리·운영 (조직관리자 전용)
- 대시보드 형태 (카드 + 차트)
- 매출/비용: 막대 차트 (월별/분기별/연도별 토글)
- 외근직 실적: 테이블 + 개인별 수치
- 내근직 업무 부하: 게이지 또는 히트맵
- 업무 지시 게시판: 기한 표시, 긴급/일반 구분

### 4-9. RAG 검색 페이지 (`/rag-search`)
```
[검색창]
"조적공 근골격계 승인 사례 찾아줘"  [검색]

[검색 범위 필터] 전체 / 승불자료 / 지사운영자료 / 지침·논문
[상병 필터] 소음성난청 / 근골격계 / 진폐 / 전체

[결과 영역]
  📋 AI 요약 (Claude API 생성)
  📄 관련 파일 (NAS 경로 + 열기 버튼)
```

## 5. 컴포넌트 스타일 가이드

### 버튼
- Primary: 스카이 블루 배경, 흰색 텍스트
- Secondary: 흰색 배경, 딥 그린 border, 딥 그린 텍스트
- Danger: 레드 배경, 흰색 텍스트
- Disabled: 그레이, 클릭 불가

### 배지 (Badge)
- 작은 pill 형태
- 색상: 상태별 컬러 규칙 따름

### 테이블
- 헤더: 딥 그린 또는 연한 그레이 배경
- 행 hover: 연한 스카이 블루 배경
- 짝수 행: 연한 그레이 배경 (zebra stripe)

### 모달
- 오버레이: `rgba(0,0,0,0.5)`
- 배경: 흰색, 적당한 border-radius
- 닫기 버튼: 우상단 X

### 폼 입력
- border: 1px solid `#CBD5E1`
- focus: 스카이 블루 border
- 에러: 레드 border + 에러 메시지

1. **기존 스타일 유지**: 인라인 스타일 / Tailwind 혼재 현황 그대로 존중
