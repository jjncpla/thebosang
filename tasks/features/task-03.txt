### 공란 인쇄 핸들러
```typescript
const handleBlankPrint = async (type: string, label: string) => {
  const res = await fetch(`/api/forms/blank?type=${type}`)
  if (!res.ok) { alert('공란 인쇄 실패'); return }
  const blob = await res.blob()
  const url = URL.createObjectURL(blob)
  const a = document.createElement('a')
  a.href = url
  a.download = `[공란]${label}.pdf`
  a.click()
  URL.revokeObjectURL(url)
}
```

### State 정의
```typescript
const [selectedForm, setSelectedForm] = useState<string | null>(null)
const [previewPage, setPreviewPage] = useState(1)
const [clickedCoord, setClickedCoord] = useState<{ x: number; y: number } | null>(null)
const [selectedField, setSelectedField] = useState<{
  key: string; label: string; x: number; y: number
} | null>(null)
const [coordFields, setCoordFields] = useState<Record<string, { key: string; label: string; x: number; y: number }[]>>({})
const [coordOutput, setCoordOutput] = useState('')
```

### 핸들러 구현
```typescript
// PDF 미리보기 클릭 → 좌표 기록
const handleCanvasClick = (e: React.MouseEvent<HTMLImageElement>) => {
  const rect = e.currentTarget.getBoundingClientRect()
  const x = Math.round(e.clientX - rect.left)
  const y = Math.round(e.clientY - rect.top)
  setClickedCoord({ x, y })
  // 선택된 필드가 있으면 해당 필드 좌표 업데이트
  if (selectedField) {
    updateFieldCoord('x', x)
    updateFieldCoord('y', y)
  }
}

// 방향키로 좌표 미세 조정
const handleArrowKey = (e: React.KeyboardEvent<HTMLInputElement>) => {
  if (!selectedField) return
  const step = e.shiftKey ? 5 : 1
  if (e.key === 'ArrowLeft')  { e.preventDefault(); updateFieldCoord('x', selectedField.x - step) }
  if (e.key === 'ArrowRight') { e.preventDefault(); updateFieldCoord('x', selectedField.x + step) }
  if (e.key === 'ArrowUp')    { e.preventDefault(); updateFieldCoord('y', selectedField.y - step) }
  if (e.key === 'ArrowDown')  { e.preventDefault(); updateFieldCoord('y', selectedField.y + step) }
}

// 좌표 업데이트
const updateFieldCoord = (axis: 'x' | 'y', value: number) => {
  if (!selectedField || !selectedForm) return
  const updated = { ...selectedField, [axis]: value }
  setSelectedField(updated)
  setCoordFields(prev => ({
    ...prev,
    [selectedForm]: prev[selectedForm]?.map(f =>
      f.key === updated.key ? updated : f
    ) ?? []
  }))
  updateCoordOutput()
}

// 좌표 전체 출력 (복사용)
const updateCoordOutput = () => {
  if (!selectedForm || !coordFields[selectedForm]) return
  const lines = coordFields[selectedForm].map(f =>
    `${f.key}: { x: ${f.x}, y: ${f.y} },`
  ).join('\n')
  setCoordOutput(lines)
}

const handleCopyCoords = () => {
  navigator.clipboard.writeText(coordOutput)
  alert('좌표가 클립보드에 복사되었습니다.')
}
```

### 서식별 필드 목록 (coordFields 초기값)
selectedForm 변경 시 해당 서식의 필드 목록을 초기화.
각 서식별 변수 목록은 task-01에서 구현한 PDF API의 변수 목록과 동일하게 구성.
초기 x, y 값은 모두 0으로 설정 (실제 좌표는 에디터에서 클릭으로 지정).

예시 (DISABILITY_CLAIM):
```typescript
{
  key: 'name', label: '성명(근로자명)', x: 0, y: 0,
  key: 'birthY1', label: '생년 1자리(Y1)', x: 0, y: 0,
  // ... 나머지 변수들
}
```

## STEP 3: 공란 인쇄 API — app/api/forms/blank/route.ts 신규 생성

```typescript
// GET /api/forms/blank?type=FORM_TYPE&preview=1(optional)&page=1(optional)
// - preview=1: PNG 이미지로 반환 (미리보기용)
// - preview 없음: PDF로 반환 (공란 인쇄용)
// 공란 PDF: 모든 변수를 빈 문자열로 치환하여 서식 구조만 출력
```

구현 방식:
- `/api/cases/[caseId]/forms/route.ts`의 각 서식 생성 함수를 재사용
- 단, 모든 변수(patient.name, dateFields 등)를 빈 문자열 "" 또는 더미값으로 대체
- 주민번호 각 자리, 날짜 각 자리 등 단일 문자 필드는 " " (공백 1자)로 대체
- Content-Type: application/pdf 반환

preview=1 처리:
- pdf-lib으로 생성한 PDF를 sharp 또는 canvas 라이브러리로 PNG 변환
- 단, sharp/canvas 설치가 필요하므로: 일단 preview=1 파라미터는 받되
  PNG 변환 없이 PDF를 그대로 반환하고, 프론트엔드에서 <embed> 또는 <iframe>으로 PDF를 표시하는 방식으로 대체
- img 태그 대신 아래로 교체:

```tsx
// 미리보기: img 태그 대신 iframe으로 PDF 직접 표시
<iframe
  src={`/api/forms/blank?type=${selectedForm}`}
  style={{ width: '100%', height: '100%', border: 'none', cursor: 'crosshair' }}
  title="서식 미리보기"
/>
// ※ iframe 위에서 클릭 좌표를 얻는 것은 보안상 제한됨
// 대신: 좌표 에디터는 "좌표 직접 입력 + 방향키 조정" 방식으로만 동작
// 클릭 좌표 기능은 향후 개선 과제로 남겨둠
// 현재는 PDF를 보면서 수동으로 x,y 값을 입력하는 방식 사용
```
