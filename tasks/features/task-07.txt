## STEP 1: schema.prisma — HearingLossDetail 필드 추가

파일: prisma/schema.prisma
HearingLossDetail 모델에 아래 필드 추가 (기존 필드 건드리지 말 것):

```prisma
// 장해급여청구서 — 수령계좌
bankName              String?   // 은행(증권사)명
bankAccount           String?   // 계좌번호
bankAccountHolder     String?   // 예금주
bankAccountType       String?   // "보통계좌" | "전용계좌"

// 장해급여청구서 — 확인사항
confirmPriorDisability  Boolean?  // ① 재해 이전 장해 여부 (true=예, false=아니오, null=미입력)
confirmPriorCompensation Boolean? // ② 동일사유 배상/보상 수령 여부
receiptDate           String?   // 수령일자 (자유 텍스트)
receiptAmount         String?   // 수령금액
receiptPayer          String?   // 지급한 자

// 장해급여청구서 — 이송비
transferCost          String?   // 이송비용 (원)
transferCostDetail    String?   // 이송비 산출내역

// 장해급여청구서 — 합병증 등 예방관리
complicationPart      String?   // 구체적 부위 또는 상병명
complicationHospital  String?   // 합병증 등 예방관리 의료기관명
```

마이그레이션:
```
npx prisma migrate dev --name add_disability_claim_fields
npx prisma generate
```

## STEP 4: PDF API 연결

파일: app/api/cases/[caseId]/forms/route.ts
DISABILITY_CLAIM 케이스에서 신규 필드 활용:

```typescript
// 수령계좌 변경 여부 체크 — 기본값 아니오([√])
// 계좌 정보가 입력된 경우 예 체크
if (detail.bankAccount) {
  drawText(page, '√', 예_위치_x, 예_위치_y, font, 9)
} else {
  drawText(page, '√', 아니오_위치_x, 아니오_위치_y, font, 9)
}

// 은행명
drawText(page, detail.bankName ?? '', bankName_x, bankName_y, font, 9)

// 계좌번호
drawText(page, detail.bankAccount ?? '', bankAccount_x, bankAccount_y, font, 9)

// 예금주
drawText(page, detail.bankAccountHolder ?? '', holder_x, holder_y, font, 9)

// 계좌종류 체크
if (detail.bankAccountType === '전용계좌') {
  drawText(page, '√', 전용계좌_x, 전용계좌_y, font, 9)
} else {
  drawText(page, '√', 보통계좌_x, 보통계좌_y, font, 9)
}

// 확인사항① 체크
drawText(page, detail.confirmPriorDisability ? '√' : '', 확인1예_x, 확인1예_y, font, 9)
drawText(page, !detail.confirmPriorDisability ? '√' : '', 확인1아니오_x, 확인1아니오_y, font, 9)

// 확인사항② 체크
drawText(page, detail.confirmPriorCompensation ? '√' : '', 확인2예_x, 확인2예_y, font, 9)
drawText(page, !detail.confirmPriorCompensation ? '√' : '', 확인2아니오_x, 확인2아니오_y, font, 9)

// 수령내역 (② 예인 경우)
if (detail.confirmPriorCompensation) {
  drawText(page, detail.receiptDate ?? '', receiptDate_x, receiptDate_y, font, 8)
  drawText(page, detail.receiptAmount ?? '', receiptAmount_x, receiptAmount_y, font, 8)
  drawText(page, detail.receiptPayer ?? '', receiptPayer_x, receiptPayer_y, font, 8)
}

// 이송비
drawText(page, detail.transferCost ?? '', transferCost_x, transferCost_y, font, 9)
drawText(page, detail.transferCostDetail ?? '', transferDetail_x, transferDetail_y, font, 8)

// 합병증
drawText(page, detail.complicationPart ?? '', compPart_x, compPart_y, font, 8)
drawText(page, detail.complicationHospital ?? '', compHosp_x, compHosp_y, font, 8)
```

※ 좌표값(_x, _y)은 /forms 에디터에서 별도 조정 예정이므로 일단 0으로 설정.
