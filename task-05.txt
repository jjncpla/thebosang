## 개요
원본 공란 PDF를 배경으로 로드하고, 그 위에 텍스트만 좌표로 삽입하는 방식으로 전환.
pdf-lib 좌표계: 좌측 하단 원점 (0,0), y축 위쪽으로 증가, A4: 595 x 841 pt

## STEP 2: lib/pdf/formUtils.ts — 오버레이 로더 추가

기존 파일에 아래 함수 추가:

```typescript
import * as fs from 'fs'
import * as path from 'path'

// 공란 PDF 로드 (오버레이용)
// fileName: 'disability_claim.pdf' 등 public/forms/ 내 파일명
export async function loadBlankForm(fileName: string): Promise<PDFDocument> {
  const filePath = path.join(process.cwd(), 'public', 'forms', fileName)
  const fileBuffer = fs.readFileSync(filePath)
  const pdfDoc = await PDFDocument.load(fileBuffer)
  return pdfDoc
}
```

```typescript
// 공란: 원본 PDF를 그대로 로드해서 반환 (텍스트 삽입 없음)
const pdfDoc = await loadBlankForm(getFileName(type))
const pdfBytes = await pdfDoc.save()
return new Response(pdfBytes, {
  headers: {
    'Content-Type': 'application/pdf',
    'Content-Disposition': 'inline',
  }
})
```
