### 1.4 시행착오 — 큰 이슈 4건

#### ① 한국어 파일명 인코딩 (Phase 1 초기)
- `subprocess.run`에서 한국어 PDF 파일명 → `WinError 2`
- 해결: PDF는 stdin/stdout 파이프, HWP는 임시 ASCII 파일 복사

#### ② pdftotext.exe 경로 오류
- 코드: `Local\Programs\poppler\` (잘못)
- 실제: `WinGet\Packages\oschwartz10612.Poppler_*\poppler-25.07.0\Library\bin\`
- 해결: 정확한 winget 경로로 수정

#### ③ Microsoft Store Python AppContainer 가상화 ⚠️ (가장 큰 시행착오)
- 1차 OCR 시도: 965/2046 모두 실패 (`Error opening data file kor.traineddata`)
- 원인: Bash의 `python` → Microsoft Store launcher → AppContainer 가상화로 `AppData\Local\Tesseract-OCR` 폴더 차단
- 진단: `os.path.exists()` False 반환, PowerShell `Test-Path` True 반환 (가상화 동작)
- 해결: tessdata를 `C:\Users\jjakg\tessdata\`로 이동 + native Python 경로(`AppData\Local\Python\pythoncore-3.14-64\python.exe`) 직접 호출

#### ④ Tesseract leptonica 메모리 누수 (Phase 3 후반)
- 5.5시간 OCR 후 1455건 성공 → 이후 290건 모두 충돌 (0xC0000005, `pixdata_malloc fail`)
- 원인: 임시 디렉토리 992개 누적 + leptonica 메모리 누수
- 영향: 우선순위 큐로 1·2순위 양식은 모두 처리 완료 → 학습 영향 없음
