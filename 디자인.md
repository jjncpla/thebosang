## 1. 브랜드 컬러 팔레트

로고에서 추출한 3가지 원색을 기반으로 시스템 전체 컬러를 구성한다.

### Primary — 스카이블루 (로고 원색 `#29ABE2`)
```
--blue-700: #1480B0   /* 진한 강조 */
--blue-600: #1A95C8   /* 호버 */
--blue-500: #29ABE2   /* 로고 원색 — Primary 버튼, 포커스 링 */
--blue-400: #50BDEA   /* 사이드바 활성 라인, 아이콘 */
--blue-300: #84D1F2
--blue-100: #DCEEFA   /* 배경 틴트, 뱃지 배경 */
--blue-50:  #EEF7FD   /* 테이블 hover, 연한 배경 */
```

### Secondary — 딥그린 (로고 원색 `#006838`)
```
--dkgreen-800: #004025
--dkgreen-700: #005530   /* 헤더 타이틀, 재해자명 */
--dkgreen-600: #006838   /* 로고 원색 — 사이드바, 페이지네이션 active */
--dkgreen-500: #00854A   /* hover */
--dkgreen-100: #D0EAD9   /* 뱃지 배경 */
--dkgreen-50:  #E8F5EE
```

### Accent — 라임그린 (로고 원색 `#8DC63F`)
```
--green-700: #5A8A1F
--green-600: #6EA02A   /* hover */
--green-500: #8DC63F   /* 로고 원색 — 신규등록·긍정 액션 버튼, 승인완료 */
--green-400: #A2D158
--green-100: #E8F5D0   /* 뱃지 배경 */
--green-50:  #F4FAE8
```

### Semantic (상태 컬러)
```
경고/이의제기: #E07010 (bg: #FEF0E0)
위험/불승인:   #C0392B (bg: #FDEDEC)
주의/서류준비: #B45309 (bg: #FEF3C7)
```

### Neutrals
```
--gray-900: #1A1F1A   /* 본문 텍스트 */
--gray-700: #374151   /* 서브 텍스트 */
--gray-600: #4B5563   /* 레이블 */
--gray-500: #6B7280   /* placeholder, muted */
--gray-400: #9CA3AF   /* 비활성 아이콘 */
--gray-300: #D1D5DB   /* 보더 강조 */
--gray-200: #E5E7EB   /* 기본 보더 */
--gray-100: #F3F4F6   /* 테이블 헤더 배경 */
--gray-50:  #F9FAFB   /* 페이지 배경 */
--white:    #FFFFFF
```

## 2. 레이아웃 구조

```
--sidebar-width:  220px
--header-height:  56px
--content-padding: 22px 24px
--radius-sm: 4px    /* 버튼, input */
--radius-md: 6px    /* 카드, 패널, 테이블 */
```

**메인 구성:** 헤더(고정) + 콘텐츠(스크롤)

## 3. 컴포넌트 가이드

### 3-1. 사이드바

- 배경: `--dkgreen-600` (#006838)
- 텍스트: `rgba(255,255,255,0.65)` — 비활성
- 텍스트 active: `white`, font-weight 600
- **활성 메뉴 라인:** 좌측 3px, `--blue-400` 색상 (브랜드 블루)
- 섹션 타이틀: `rgba(255,255,255,0.35)`, uppercase, letter-spacing 1.1px
- 알림 뱃지: `--blue-500` (기본) / `--green-600` (긍정)
- hover: `rgba(255,255,255,0.07)` 배경

### 3-2. 헤더

- 배경: white, 하단 border `--gray-200`
- 페이지 타이틀: `--dkgreen-700`, font-size 15px, font-weight 700
- 브레드크럼: `--gray-400`, font-size 12px

### 3-3. 버튼

| 종류 | 배경 | 용도 |
|------|------|------|
| `btn-primary` | `--blue-500` | 검색, 조회, 일반 확인 |
| `btn-accent` | `--green-500` | 신규등록, 저장, 긍정 액션 |
| `btn-secondary` | `--dkgreen-600` | 제출, 중요 처리 |
| `btn-outline` | white + border | 취소, 내보내기, 상세보기 |
| `btn-danger` | `#C0392B` | 삭제 |

- 기본 높이: 34px / 소형: 28px (`btn-sm`)
- border-radius: `--radius-sm` (4px)
- font-weight: 600

### 3-4. 입력 폼

- 높이: 34px
- border: 1px solid `--gray-200`
- focus: border `--blue-500`, box-shadow `rgba(41,171,226,0.14)` 3px
- label: font-size 11px, font-weight 600, color `--gray-600`

### 3-5. 카드 / 패널

- background: white, border: 1px `--gray-200`, border-radius: `--radius-md`
- box-shadow: `--shadow-sm`
- **상단 컬러 라인 (stat-card용):**
  - 파랑: `--blue-500`
  - 딥그린: `--dkgreen-600`
  - 라임그린: `--green-500`
  - 경고: `#E07010`

### 3-6. 테이블

- 헤더: background `--gray-50`, font-size 11px, uppercase, `--gray-500`, letter-spacing 0.4px
- 행 hover: `--blue-50` (#EEF7FD)
- 행 구분선: `--gray-100`
- 재해자명: font-weight 600, color `--dkgreen-700`
- 부가 텍스트 (번호·날짜): `--gray-500`, font-size 12px

### 3-7. 상태 뱃지 (badge)

| 상태 | 배경 | 텍스트 |
|------|------|--------|
| 접수중 | `--blue-100` | `--blue-700` |
| 진행중 | `--dkgreen-100` | `--dkgreen-600` |
| 승인완료 | `--green-100` | `--green-700` |
| 이의제기중 | `#FEF0E0` | `#C45E00` |

| 불승인 | `#FDEDEC` | `#C0392B` |

- padding: 2px 8px, border-radius: 20px, font-size 11px, font-weight 600

### 3-8. 상병유형 태그 (disease-tag)

| 상병 | 배경 | 텍스트 |
|------|------|--------|
| 소음성 난청 | `--blue-100` | `--blue-700` |

## 5. 전역 CSS 변수 적용 위치

`app/globals.css` 또는 `styles/globals.css`의 `:root` 블록에 전체 변수 선언.  
Tailwind 사용 시 `tailwind.config.ts`의 `theme.extend.colors`에 브랜드 컬러 등록.

```css
/* tailwind.config.ts — colors 확장 예시 */
colors: {
  blue: {
    50: '#EEF7FD', 100: '#DCEEFA',
    300: '#84D1F2', 400: '#50BDEA',
    500: '#29ABE2', 600: '#1A95C8', 700: '#1480B0',
  },
  dkgreen: {
    50: '#E8F5EE', 100: '#D0EAD9',
    500: '#00854A', 600: '#006838',
    700: '#005530', 800: '#004025',
  },
  green: {
    50: '#F4FAE8', 100: '#E8F5D0',
    400: '#A2D158', 500: '#8DC63F',
    600: '#6EA02A', 700: '#5A8A1F',
  },
}
```

## 6. 디자인 원칙

1. **브랜드 통일성** — 홍보물·로고와 동일한 3색 체계 유지
2. **눈 피로 최소화** — 쨍한 원색 직접 배경 사용 금지. 배경은 gray-50/white, 원색은 포인트·버튼·라인에만
3. **전문성** — 과도한 애니메이션·장식 배제. 정보 밀도와 가독성 우선
4. **상태 명확성** — 사건 상태는 반드시 배지 컬러로 즉시 식별 가능하게
5. **일관성** — 동일 의미의 컴포넌트는 전 페이지 동일 스타일 적용
